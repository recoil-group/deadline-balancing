Shotgun spread

A small interactive demo of Deadline 0.25.2-style shotgun spread math. Defaults are the stock R870 00 buck setup using buck_barrel_deviation = 2.55, spread_multiplier = 1.5, spread_factor = 0.8, and 9 projectiles.

Game formula

input = use_buck_deviation ? buck_barrel_deviation : barrel_deviation max = tan(input / 100) * spread_multiplier dir = Vector2(rand(-1, 1), rand(-1, 1)).Unit vec = dir * max * (rand(0, 1) ^ spread_factor) final = (barrelCFrame * CFrame.Angles(vec.X, vec.Y, 0)).LookVector
